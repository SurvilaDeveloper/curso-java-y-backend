# Java desde cero

Aplicación web hecha con **Astro** para aprender **Java** de forma progresiva, práctica y autodidacta.

Este proyecto fue creado como un espacio de estudio y consulta para personas que quieren aprender Java en serio, con una base sólida y con la posibilidad de **repasar conceptos**, **leer ejemplos** y **autoevaluarse**.

## Objetivo del proyecto

La idea de esta aplicación es servir como material de apoyo para el aprendizaje de Java y Backend, especialmente para quienes prefieren estudiar por su cuenta o complementar un curso formal.

No busca reemplazar una formación guiada por docentes o tutores, sino ofrecer un recurso accesible para:

- estudiar temas de Java de forma ordenada
- consultar contenido cuando haga falta
- practicar con ejemplos
- reforzar conocimientos mediante autoevaluación

## Características

- Sitio estático desarrollado con **Astro**
- Contenido organizado por temas
- Navegación simple y rápida
- Modal informativo del proyecto
- Favicon personalizado
- Base visual oscura y moderna
- Preparado para crecer con más módulos y contenidos
- Proyecto público y editable

## Tecnologías utilizadas

- **Astro**
- **TypeScript**
- **CSS**
- **Markdown** para el contenido de los temas

## Filosofía

Este proyecto está pensado desde una lógica de aprendizaje práctico: teoría suficiente para entender bien cada tema, pero con foco en avanzar, construir criterio y ganar soltura programando.

También está publicado como **software libre**, para que cualquier persona pueda:

- leer el código
- adaptarlo a sus necesidades
- reutilizar ideas
- mejorarlo
- crear su propia versión

## Estado del proyecto

El proyecto se encuentra en desarrollo y puede seguir creciendo con nuevos temas, mejoras visuales, más ejercicios y nuevas secciones relacionadas con Java y Backend.

## Instalación y ejecución

Cloná el repositorio:

```bash
git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git
```

Entrá en la carpeta del proyecto:

```bash
cd TU_REPOSITORIO
```

Instalá las dependencias:

```bash
npm install
```

Iniciá el entorno de desarrollo:

```bash
npm run dev
```

Luego abrí la URL local que te muestre Astro en la terminal.

## Build de producción

Para generar la versión lista para producción:

```bash
npm run build
```

Para previsualizar el resultado del build:

```bash
npm run preview
```
